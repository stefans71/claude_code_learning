# claude-code-learn

You are inside a self-teaching Claude Code tutorial. This repo teaches
Claude Code features by BEING an example of them.

## What this repo is
A set of skill files and lesson content. No code. No scripts. No executables.
Claude Code is the installer, the runtime, and the operator.

## Available commands
- /setup — Run this first. I'll interview you and build your learning environment.
- /lesson [topic] — Start a guided lesson (e.g., /lesson 01-CLAUDE-md)
- /explain — Break down what just happened mechanically
- /challenge — Hands-on task harder than the lesson exercise
- /progress — Show what you've covered and what's next
- /UDA [topic] — Full Use → Demonstrate → Apply learning loop

## How this works
Each lesson teaches a Claude Code feature by pointing at this repo's own
configuration as the example. The CLAUDE.md you're reading right now IS
lesson 01. The skills that drive /lesson and /explain ARE lesson 03.
The hooks (once /setup creates them) ARE lesson 04.

## Rules for this session
- Always explain what you're about to do and why before doing it
- After completing an action, note one thing the learner should observe
- Reference this repo's own files as examples whenever possible
- If the learner seems confused, switch to Socratic questions
- When the learner creates a file, read it and give specific pass/fail feedback
